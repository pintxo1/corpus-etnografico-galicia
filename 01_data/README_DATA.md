# Data Guide

This repository does NOT include the full TEI corpus.

## Where to place TEI

Put TEI XML files here:

- 01_data/tei/source/

The pipeline expects TEI files with .xml or .tei.xml extensions.

## Sample data

A placeholder is provided in:

- 01_data/tei/sample/

## External data

If you use external archives (IGE, newspapers, etc), place them under:

- 01_data/external/

Do not commit large or restricted files to git.
